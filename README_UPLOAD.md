# Minimal Upload Pack for Nilavra.github.io

This is the smallest stable version of the site.

## Files included

- `index.html`
- `style.css`

## Important

Your repo should already have:

```text
assets/profile-pic.jpg
```

The homepage uses exactly this image path:

```html
<img src="assets/profile-pic.jpg" alt="Nilavra Pathak profile photo">
```

## Upload steps

1. Open your GitHub repo: `Nilavro/Nilavra.github.io`
2. Upload `index.html` and `style.css` to the repo root.
3. Choose **Replace existing files** when GitHub asks.
4. Keep your existing `assets/profile-pic.jpg`.
5. Hard refresh the live page after GitHub Pages finishes deploying.

## Optional cleanup

Since this is the minimal static version, you can delete these old Jekyll files if you want:

- `index.md`
- `_config.yml`
- `_layouts/`
- `_includes/`
- `_research/`
- `_posts/`

But you do not have to delete them if `index.html` is present and correct.
